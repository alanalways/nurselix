## ✍️ 出題流程（Narrative-First，嚴格遵守）

**每一題都必須按此順序思考，禁止跳步驟：**

### Step 1：先用中文寫一段真實臨床故事
在寫任何選項之前，先完整地用中文構思病患敘事：
- 病患年齡、性別、主要疾病
- 具體且真實的生命徵象數值（不可憑空捏造，需符合該病理狀況）
- 護理師在什麼情境下觀察到什麼
- 正在發生什麼問題，問題的嚴重程度

### Step 2：從敘事中提取出題重點
- 這題考的是什麼護理核心概念？
- 正確答案的生理/藥理/法律機轉是什麼？
- 每個干擾選項的「為什麼錯」需要是真實的醫學錯誤原因

### Step 3：轉換為 JSON 格式輸出

---

## ❌ 絕對禁止的 Mad Libs 反模式

以下任何一條觸發，該題視為污染題，必須重寫：

1. **題幹是填空表格**：`"37F lisinopril use: BP 97/57, HR 104"` 這種縮寫填空 → 禁止
2. **解析開頭重複**：所有題的 `explanationZh` 不得以同一句話開頭
3. **usTwDifference 輪迴**：同一批次內，台美差異說明不可出現完全重複的內容
4. **選項理由是廢話**：`"因為它會讓目前問題更惡化"` 沒有講明機轉 → 禁止
5. **數值不合臨床邏輯**：血壓 82/48 的病患不應該是「EASY 難度」的題目
6. **選項結構雷同**：4 個選項的第一個字都一樣（如都是「給予...」）

  "difficulty": "EASY 或 MEDIUM 或 HARD",

## 🎯 臨床場景多樣性要求

每批 150 題中，臨床場景需涵蓋：
- **病房類型**：ICU、急診、術後恢復室、門診、居家護理、精神科、長照、社區至少各出現 1 次
- **年齡層**：新生兒/嬰幼兒、學齡兒童、成人、老年人（>65）都要出現
- **疾病系統**：心血管、呼吸、神經、腸胃、腎臟、內分泌、感染、腫瘤至少各 1 題
- **護理角色**：優先排序、委派/delegating、病患衛教、倫理、法律、跨團隊合作都要有題

## ✅ 每批輸出前的自我檢查清單

生成完 150 題後，輸出之前請逐條確認：

- [ ] 所有題的 `explanationZh` 開頭是否各不相同？
- [ ] 同批次 `usTwDifference` 是否沒有任何完全重複？
- [ ] 所有 MCQ 的 `correctAnswers` 是 `null`？
- [ ] 所有 SATA 的 `correctAnswer` 是 `null`，且 `correctAnswers` 是陣列？
- [ ] 所有題都有 `irtA`、`irtB`、`irtC`？
- [ ] 沒有題幹使用「37F」「72M」這類填空式縮寫開頭？
- [ ] 沒有任何 `optionE` 出現在 MCQ 中？
- [ ] 最後 3 題的選項說明是否仍然完整不亂碼？

直接給 JSON，其他什麼都不要。
